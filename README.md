# FranceReconciliator
Applicazione desktop Python per autenticazione OAuth OCI e ricerca fornitore.

## Setup ambiente virtuale

```bash
cd /Users/mvarisco/vibecode/FranceReconciliator/FranceReconciliator
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

In alternativa esegui lo script:

```bash
cd /Users/mvarisco/vibecode/FranceReconciliator/FranceReconciliator
bash setup_env.sh
```

## Avvio dell'app

```bash
source .venv/bin/activate
python3 main.py
```
